General Information

Version List

  • NTSC - U/C : Dante takes 20% more damage. Controls are different △ is jump instead of melee, O is melee instead of inspect, and X is inspect instead of jump
  • NTSC - J : Same as US version however, Control is the standard DMC default
  • PAL : runs at 50Hz which is slower than NTSC (60Hz)
  • HD Collection 2012: Missing effects from PS2
  • HD Collection 2018 : Griffon's chest won't open for some reason

Timing

Timed by Loadless.

  • Time starts when you select difficulty (the "Devil May Cry" audio cue will play).
  • Time ends when you skip the plane cutscene.
  • Total Results screen at the end of the run (screen includes IGT, orb count, and mission rankings) must be included with the submission.

Categories

  • Any% - Beats the game as fast as possible.
  • All Collectibles - All Blue orbs/fragments, weapons, secret missions must be completed before beating the game
  • Low% - Beats the game without upgrades or use of weapons that aren't mandatory.
